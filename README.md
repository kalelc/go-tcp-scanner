# TCP Scanner in Golang

### Example Use

```
go build -o scanner
go run scanner
```

### Scanner Help

```
./scanner --help
Usage of ./scanner:
  -end-port int
    	the end port to scan (default 100)
  -hostname string
    	hostname to scan ports (default "localhost")
  -start-port int
    	the start port to scan (default 1)
```
